Patient Login Access

The Add New User section allows you to add new patient and do all the registion process through the appliction.

  • Login: Enter the Patient's registered email to access the patient portal.
  • Password: Enter the password corrosponding to the registered email.

Add New Patient

The Add New User section allows you to add new patient and do all the registion process through the appliction.

  • First Name: Enter the user's first name for identification and communication.
  • Middle Name: Provide the user's middle name to differentiate between similar names.
  • Last Name: Enter the user's last name for formal identification and record-keeping.
  • Date of Birth: Input the user's date of birth for age verification and demographic purposes.
  • Medicare Number: Include the user's Medicare number for health insurance and medical records.
  • Western Affairs Number: Add the Western Affairs number if applicable for administrative purposes.

Troubleshooting

Having trouble with the system? Check out our troubleshooting guide below to resolve common issues.

  • Login Issues:

    Ensure you have the correct username and password. If you forget your password, use the "Forgot Password" link.

  • Check for Sensor Issues:

    If the activity data seems inconsistent or missing, the sensors responsible for tracking movements might be malfunctioning..

  • Activity Data Not Loading:

    Refresh the page or check your internet connection.
