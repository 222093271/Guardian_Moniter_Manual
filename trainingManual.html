Welcome to Training Manual for Guardian Monitor Admin Page

Introduction

The purpose of this manual is to guide administrators on effectively using the new admin page design for Guardian Monitor.

The audience for this manual includes new and existing administrators of Guardian Monitor.

This manual provides an overview of the new design and its features, ensuring smooth navigation and utilization.

  • Login Page: The login page provides a secure access to the admin page.
  • Username and Password: Enter your assigned username and password to access the admin dashboard.

Dashboard Overview

The dashboard is the central hub where you can access various features and monitor system activities.

  • Main Dashboard Features: The main dashboard features include activity overview, user statistics, and quick links.
  • Navigation Menu: The navigation menu provides easy access to different sections of the admin page.

User Management

The user management section allows you to add, edit, and delete users.

  • Add User: To add a new user, click on the "Add User" button and fill in the required details.
  • Edit User: To edit an existing user's information, select the user and click on the "Edit" button.
  • Delete User: To delete a user, select the user and click on the "Delete" button.

Activity Monitoring

The activity monitoring section allows you to view and monitor system activities.

  • View Activity Data: To view activity data, select the time period and click on the "View" button.
  • Generate Reports: To generate reports, select the report type and click on the "Generate"

Troubleshooting

Having trouble with the system? Check out our troubleshooting guide below to resolve common issues.

  • Login Issues:

    Ensure you have the correct username and password. If you forget your password, use the "Forgot Password" link.

  • User Management Errors:

    Check for correct user details and roles.

  • Verify the Date and Time:

    Make sure the "Last Updated" date and time are correct. If not, it might mean the system hasn't properly recorded recent activity. Try refreshing the page or logging out and back in to see if it updates the data.
